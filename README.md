# Proyecto Integrador AWS — Cloud Infrastructure & Integration Documentation

This repository houses the complete planning, development, implementation, and final documentation guides for migrating and integrating our system architectures onto Amazon Web Services (AWS).

## Documentation Index

Please consult the following documents for specific phases and tools within our cloud integration roadmap:

1. **[Desarrollo del proyecto integrador](Desarrollo%20del%20proyecto%20integrador.md)**: Details the active deployment stages, architecture diagrams, cloud integrations, and execution phases.
2. **[Documentación Final del Proyecto](Documentacion_Final_Proyecto.md)**: The final submission draft summarizing the overall integration architecture, achievements, resource optimization, and final review points.
3. **[Instrucciones Jules AWS](Instrucciones_Jules_AWS.md)**: Configuration guides, script runbooks, and credentials templates optimized for virtual environments on AWS.
4. **[AWS CLI Implementation Plan](aws_cli_implementation_plan.md)**: Step-by-step terminal guides for configuring IAM roles, S3 buckets, EC2 nodes, database connectors, and security protocols via the AWS Command Line Interface.

## Key Goals

- **Highly Available & Scalable Architecture**: Infrastructure blueprints utilizing VPCs, Subnets, Security Groups, and multi-node distributions.
- **Automated Deployments**: AWS CLI runbooks detailing standard execution commands to quickly replicate setups.
- **System Hardening**: Integrated security rules, access controls, and private data subnets.
